# Pilot Charter

## Pilot Objective

Evaluate whether an internal AI knowledge assistant can reduce search time and improve employee access to approved internal knowledge while maintaining appropriate human oversight.

## Pilot Scope

- One team
- One workflow
- Approved knowledge base
- Limited question categories
- Human escalation path

## Success Criteria

- Users return after first use
- Accepted or edited responses exceed rejected responses
- High-risk cases are escalated
- Source gaps are identified
- Users report time saved

## Exit Criteria

The pilot should not scale if:

- users do not return
- high-risk questions are mishandled
- source quality is poor
- business value is not measurable
- escalation ownership is unclear
