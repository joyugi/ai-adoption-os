# User Journey

## User Goal

Get a fast, reliable answer to an internal policy or process question.

## Journey

1. User has a question during a workflow.
2. User asks the AI assistant.
3. Assistant searches approved knowledge.
4. Assistant provides an answer, source, confidence label, and risk label.
5. User decides to accept, edit, reject, or escalate.
6. System logs adoption telemetry.
7. Product team reviews patterns to improve the assistant.

## Key Trust Moments

- Does the user understand where the answer came from?
- Does the answer include the right level of caution?
- Does the user know when not to rely on the assistant?
- Is escalation easy?
- Is feedback captured without disrupting the workflow?
